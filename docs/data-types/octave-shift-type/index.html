Octave shift type data type

Base type: Integer (signed)

Allowed values

-15 15ma (notes are rendered down two octaves)
-22 22ma (notes are rendered down three octaves)
-8 8va (notes are rendered down one octave)
15 15mb (notes are rendered up two octaves)
22 22mb (notes are rendered up three octaves)
8 8vb (notes are rendered up one octave)

Where is this type used?

<octave-shift> — "type" attribute
