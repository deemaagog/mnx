The <tempo> element

Parent element: <directions-global>

The <tempo> element defines a tempo, from the point of its occurrence forward until changed. The tempo is rendered as a conventional pairing of a small beat unit notation equated to a metronome marking.

Content

Always empty.

Attributes

Name Type Required? Description
bpm Floating point number Yes The number of times per minute the given value should occur.
value Note value Yes A note value which is asserted to occur with a frequency of bpm times per minute.
location Measure location No The measure location of this tempo marking. If not provided, the default value is zero.
